    # 项目二：教务蜂巢管理系统

    • 技术栈：Vue 3 + Element Plus + Vite + TailwindCSS + Axios + Vuex + Vue Router + Express + MySQL + JWT + Node.js
    • 项目描述：一个面向学校教务管理的综合系统，实现了学生信息管理、课程安排、成绩管理、考勤统计等核心功能。采用前后端分离架构，前端使用Vue 3全家桶构建响应式管理界面，后端基于Express提供RESTful API接口，通过JWT实现用户认证和权限控制。
    • 我的职责：独立完成全栈开发，包括：
      • 前端：使用Vue 3 + Element Plus搭建管理界面，实现数据可视化展示（ECharts图表）、国际化（vue-i18n）、组件自动化导入等
      • 后端：基于Express构建RESTful API，设计数据库模型，实现用户认证、数据接口、文件上传等功能
      • 数据库：使用MySQL进行数据存储，设计合理的表结构和关系
      • 部署：配置开发环境，实现前后端联调测试
    • 项目亮点：
      • 采用Vite构建工具，提升开发效率和构建速度
      • 集成Element Plus UI组件库，快速搭建美观的管理界面
      • 使用JWT实现安全的用户认证机制
      • 引入ECharts实现数据可视化，直观展示统计信息
      • 通过Vuex进行全局状态管理，配合持久化插件保证数据一致性
      • 前后端分离架构，提高系统可维护性和扩展性
      • 使用TailwindCSS实现响应式布局，适配不同屏幕尺寸
